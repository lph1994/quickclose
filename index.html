💡 How to use QiClose

Get started in under a minute — follow these steps.

1

Type the program name. When the suggestions box appears, click the one you want to fill in the yellow input field.

2

Click the + button to add the program (e.g., Firefox) to the yellow note on the right side.

3

Repeat Steps 1 & 2 for other apps (e.g., Telegram, Word...). Then click "Hide" to minimize to the system tray:

4

Press the shortcut key: Ctrl + Alt + Backspace or ALT + 1

✓

Done! Your app list is saved. From now on, just press Ctrl + Alt + Backspace and your screen and RAM are instantly clear.

💬 Use QiClose With ChatGPT

Can't find your app in the list? Ask ChatGPT for the .exe name and add it.

1

Click the GPT button inside QiClose.

2

When ChatGPT opens in your browser, type: "What exe file name of [your application]?"
Example: "What exe file name of CapCut PC" → Answer: Capcut.exe

3

Copy Capcut.exe into the input field and click Add. CapCut is now in your QuickClose list.

✓

Done! Press Ctrl + Alt + Backspace — it will close together with all other apps in your list.

⚠️ Important Warning

🚨 Back up your data before using QiClose to close all apps.
📝 Avoid adding system or security applications — closing them may cause errors or affect system stability.
💻 Only use when your PC is slow, frozen, or RAM-overloaded — not as a daily habit.
🚫 This tool is powerful but cannot replace manual app management in all situations.
👨‍💻 Best suited for tech professionals — developers, testers, debuggers, or miners who understand system processes.
